# macOS和IOS常用软件

## 效率

* picgo：图片拖拽上传，剪切板快捷键上传，按上传文件名覆盖
* ScrollReverser: 触摸板和鼠标个性化配置。
* screenkey 屏幕回显按键，录制视频时使用。
* balsamiq Mockups: 原型绘图软件。

* AppZapper

* FileZilla

* Microsoft Remote Desktop:  mac访问Windows
* Apple remote desktop / nomachine /nuords :  mac远程
